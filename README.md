# OverTheWire: Bandit Wargame Writeups

## 📌 About
This repository contains my documentation and walkthrough of the Bandit wargame from OverTheWire.

[OverTheWire’s Bandit](https://overthewire.org/wargames/bandit/) is an interactive wargame designed to teach command-line concepts and foundational cybersecurity skills. Each level presents a challenge where the goal is to retrieve the password for the next level by exploring Linux commands and tools. 


Bandit focuses on:
- Linux fundamentals
- File permissions
- SSH
- Encoding & decoding
- Archive extraction
- Basic scripting

## 🎯 Goal
To strengthen Linux command-line skills and build a strong foundation in cybersecurity.

## 📂 Structure
Each level is documented separately inside the `levels/` directory.


Each `levels/LevelXX.md` file contains:

- **Objective**
- **Concept used**
- **Commands used**
- **Explanation**
- **Key takeaways / lessons**
